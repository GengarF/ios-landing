# Swift Basics Day 21

## 今天学了什么
- 第一个纯 Swift 命令行小项目
- 多文件组织
- 简单数据模型设计
- Store 管理数据
- private 封装内部数组
- map / filter / reduce 在小项目中的使用
- 格式化输出 helper

## 核心理解
- 小项目不应该把所有代码堆在主文件里
- Model 负责表达数据
- Store 负责管理数据
- Helper 负责辅助格式化
- Main 文件负责组装流程
- private 可以保护内部状态
- 高阶函数可以让数据处理更清楚

## 今天最重要的收获
- 我开始从单题练习过渡到小项目组织
- Student / GradeRecord / GradeStore 已经有一点真实工程分层的感觉
- Swift 代码开始不只是“能跑”，还要考虑职责划分
