# Swift Basics Day 18

## 今天学了什么
- 存储属性
- 方法
- mutating
- 计算属性
- static
- 属性、方法、计算属性综合使用

## 核心理解
- 存储属性负责保存状态
- 计算属性负责给出派生结果
- 方法负责表达行为
- mutating 表示值类型的方法会修改自身
- static 属于类型本身，而不是某个实例

## 今天最重要的收获
- 一个模型不只是字段堆积，还可以带方法和计算属性
- 真实业务对象往往会把状态、派生信息和行为组合起来
- VideoProgress 这种类型已经有一点真实业务模型的感觉了

- 一些练习里的数值可以更贴近学习场景
